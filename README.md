# Loopin AI

AI layer for Loopin: local small models, LoRA/QLoRA fine-tuning, tool calling, event recommendation, and hybrid Gemini fallback.

## Goals
- Extract user interests
- Route user requests to tools
- Generate structured JSON tool calls
- Recommend events and event buddies
- Run on low-resource CPU servers
